## zodiak.js
Kode program ini dibuat menggunakan **Node.js** untuk menentukan zodiak seseorang berdasarkan tanggal dan bulan kelahiran yang dimasukkan melalui terminal. Pada bagian awal, `const readline = require("readline");` digunakan untuk mengimpor modul bawaan Node.js bernama **readline** yang berfungsi membaca input dari pengguna melalui terminal. Selanjutnya dibuat variabel `rl` dengan `readline.createInterface()`. Bagian `input: process.stdin` berfungsi untuk menerima data yang diketik oleh pengguna melalui keyboard, sedangkan `output: process.stdout` berfungsi untuk menampilkan teks atau pertanyaan ke layar terminal. Beberapa `console.log()` digunakan untuk menampilkan tampilan pembuka program. Bagian ini berfungsi sebagai tampilan awal yang menjelaskan tujuan program, yaitu untuk mengetahui zodiak berdasarkan tanggal lahir. Garis pemisah menggunakan tanda `=` dibuat agar tampilan lebih rapi dan mudah dibaca. Program kemudian menggunakan `rl.question()` untuk meminta input dari pengguna. Pertanyaan pertama meminta nama pengguna dan hasilnya disimpan dalam parameter `nama`. Setelah itu, program kembali memanggil `rl.question()` untuk meminta tanggal lahir dan menyimpannya dalam parameter `tanggal`. Pertanyaan ketiga meminta bulan lahir dan menyimpannya dalam parameter `bulan`. Nilai `tanggal` dan `bulan` yang awalnya bertipe string kemudian diubah menjadi angka menggunakan `parseInt()`. Hal ini penting agar data dapat dibandingkan menggunakan operator logika seperti `>=`, `<=`, dan `===`. Selanjutnya dibuat variabel `zodiak` untuk menyimpan hasil penentuan zodiak. Proses penentuan zodiak dilakukan menggunakan struktur percabangan `if...else if...else`. Setiap kondisi memeriksa kombinasi bulan dan tanggal tertentu. Misalnya:
Jika bulan Januari dan tanggal lebih dari atau sama dengan 20, atau bulan Februari dan tanggal kurang dari atau sama dengan 18, maka zodiaknya adalah Aquarius. Struktur logika ini diterapkan untuk seluruh 12 zodiak, yaitu Aquarius, Pisces, Aries, Taurus, Gemini, Cancer, Leo, Virgo, Libra, Scorpio, Sagitarius, dan Capricorn. Operator `&&` digunakan untuk menyatakan kondisi “dan”, sedangkan `||` digunakan untuk menyatakan kondisi “atau”. Jika tanggal atau bulan yang dimasukkan tidak sesuai dengan rentang yang valid, maka program akan menampilkan pesan error menggunakan `console.log()` dan menghentikan proses dengan `rl.close()` serta `return` agar program tidak melanjutkan eksekusi. Jika input valid, program akan menampilkan hasil akhir berupa nama pengguna, tanggal lahir, bulan lahir, dan zodiak yang sesuai. Bagian ini ditampilkan dengan format yang rapi menggunakan beberapa `console.log()`. Terakhir, `rl.close()` digunakan untuk menutup koneksi input setelah program selesai dijalankan agar terminal tidak terus menunggu input tambahan.
Secara keseluruhan, program ini menggunakan konsep dasar Node.js, input/output terminal, percabangan kondisi, serta operator logika untuk menentukan zodiak berdasarkan tanggal kelahiran pengguna.
